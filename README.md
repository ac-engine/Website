# Website

middlemanとslimによりサイトを構築しています。

middlemanはBOM無しUTF-8のみ対応なので(2015/05)、Windowsで開発している方は注意するようお願いします。

## Windowsの環境構築

### インストールするソフトウェア

```
Ruby 2.0.0-p643
DevKit-mingw64-32-4.7.2-20130224-1151-sfx.exe
```

### インストールのためのコマンドライン

```
gem install middleman
gem install bundler
bundle update
```
